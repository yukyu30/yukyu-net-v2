ゲーミングPCを買い替えました。
LLMなどや音声変換、合成を自分のPCで扱えるようにするという建前で散財しましたが、本当は9月に発売されるStarfieldというゲームをプレイするためです。

## 激動の二日間

### パーツ選定
まず、土曜日にPCを組み立てたことがある同僚と一緒に秋葉原に行き、パーツの選定をしました。

ちなみに前のPCの構成はDELLの XPS8910, グラボは交換してGTX1660Tiでした。
1660TiはCUDAが使えなくて、色々不便でした

今回、予算は30万円、ChatやStarfiledといったゲームをしたい。ということで、以下のようなパーツを選びました。

| パーツ | 値段 |
| --- | --- |
| CPU | Intel Core i7-13700K 
| GPU | GIGABYTE NVIDIA GeForce RTX 4070 EAGLE OC 12G |
| マザーボード | MSI MAG Z69 TOMAHAWK WIFI |
| メモリ | crucial DDR5 Pro 16GBx2 |
| SSD | SAMSUNG 980 PRO 2TB |
| 電源ユニット | CORSAIR RM850x |
| ケース | Fractal Design North |
| CPUクーラー | Noctua NH-U12A |
| クーラー | Noctua NF-F12 PWM chromax.black.swap x 2 |



最終的には30万1千円でした。セール品もあったので、ほとんど予算内で収まりました。

### 組み立て
日曜には同僚を家に招き、組み立てを行いました。
緊張しながらグリスを塗ったり、CPUを取り付けたりしました。

組み立て終わって、電源を入れても、画面に何も映りませんでした。
GPUをはずしてCPUの内蔵グラッフィックでは表示されたので、GPUを入れ直しました。
それでも出力されず、初期不良かなとおもったのですが、電源ユニット側のケーブルを差し直したら映像が表示されました。

## 打ち上げ
ピザを頼んで、家で打ち上げをしました。
とういうかもはや、打ち上げがメインみたいな感じになって10時間打ち上げしてました。
PC組み立て3時間、打ち上げ10時間🤔

30万のPCでニコニコ動画へ投稿した動画（黒歴史）や、Youtubeで夢グループの高圧洗浄機の動画を見てました。

## ゲーム楽しみ
Starfieldが発売されるのが9月なので楽しみです！
また、Chatも快適になると思うのでたくさんたいです！！
