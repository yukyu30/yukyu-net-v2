# M!LK とは

[公式プロフィール](https://sd-milk.com/profile)にもかいてありますが、ダンスボーカルグループです。
各メンバーはドラマや、舞台などでマルチに活躍中です。最近では[M!LK 公式 Youtube チャンネル](https://www.youtube.com/channel/UCHqQCpqvSOcHQr6Oqa4_5wg)で

# 知ったキッカケ

ドラマにメンバーの一人である、佐野勇斗さんがドラマに出演しており、気になり調べ M!LK で活動してることを知りました。2019 年くらいだと思います。その時は曲は聞いていませんでした。
2020 年になり、コロナ禍で引きこもり生活をしてるときに Youtube でたまたま[M!LK - 試しにライブを別撮りで撮ってみた](https://www.youtube.com/watch?v=zF2FnNaWyFc)という動画を見かけ、そこからはまりました。
